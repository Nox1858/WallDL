# WallDL

## Prerequisites
- Linux (Plasma Desktop), MacOS or Android (Windows and iOS support is currently not planned)
- Internet Connection (For getting images and tag data)
- Gel API access

## Getting Started
### Linux with Plasma Desktop:

1. clone the repository
2. run `python setup.py` and follow instructions
3. run `python wallpapers.py help` to see possible options
### MacOS

1. clone the repository
2. run `python setup.py` and follow instructions
    - note that you should use subfolders of WallDL since you'll probably run into permission errors otherwise
3. run `python wallpapers.py help` to see possible options
### Android

1. install [termux](https://f-droid.org/en/packages/com.termux/) (the playstore version is not tested and might not work)
2. install [termux-api](https://f-droid.org/en/packages/com.termux.api/)
3. install python `pkg install python`
4. install git `pkg install git`
5. go into some directory and clone this repository `git clone https://Nox1858/WallDL`
6. run setup `python setup.py` and follow instructions
    - note that you should use subfolders of WallDL, anything else might be annoying to get to work
    - You might need to manually create some of the folders since this step might fail, check installed desktop versions to see folder structure and necessary files
7. run `python wallpapers.py help` to see possible options
